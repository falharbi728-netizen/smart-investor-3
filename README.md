## ▶️ كيفية تشغيل الواجهة | How to Run

### الطريقة الأسهل — HTML مباشر:
1. افتح ملف `smart-investor.html` في أي متصفح حديث
2. ✅ لا يحتاج إلى تثبيت أي شيء!

### تشغيل React (للمطورين):
```bash
npx create-react-app my-app
cd my-app
# انسخ محتوى smart-investor-v2.jsx إلى src/App.jsx
npm start
```

## ✨ المميزات | Features

- 🌐 **ثنائي اللغة** — عربي كامل + إنجليزي مع RTL/LTR
- 🗺️ **خريطة تفاعلية** — عرض الفرص على خريطة المملكة
- 🔍 **بحث وفلترة متقدم** — حسب القطاع والمخاطرة والمنطقة
- 🧮 **حاسبة العائد** — احسب أرباحك بشكل ديناميكي
- 📊 **مؤشر السوق** — رسم بياني مباشر للسوق
- ⚖️ **مقارنة الفرص** — قارن حتى 3 فرص جنباً إلى جنب
- 🤖 **تحليل AI** — ملخصات ذكية للفرص بالذكاء الاصطناعي
- 💬 **المراسلات** — تواصل مع المستشارين الماليين
- 📰 **شريط أخبار** — تيكر مباشر لأخبار الاستثمار
- 📈 **مصفوفة المخاطر** — عرض مرئي للعائد مقابل المخاطرة
- 💰 **نسبة التمويل** — تتبع تمويل كل فرصة
- 🔔 **الإشعارات** — تنبيهات الفرص الجديدة
- 📱 **متجاوب** — يعمل على الهاتف والكمبيوتر
